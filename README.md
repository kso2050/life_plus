## 문화 공유 프로젝트

#### 해피투게더는 자신이 좋아하는 음악, 영화, 책 등을 자유롭게 공유하고 추천 받는 프로젝트
#### 유저들이 다양한 정보를 추가하고 추천하는 기능을 가짐.
#### 크게 회원과 음악, 영화, 책, 추천 list 로 구분

* 회원
   - 이름
   - 성별
   - 연락처
   - ID
   - PW

* 음악 
   - 노래 제목
   - MusicNum
   - 앨범
   - 아티스트

* 영화 
   - 제목
   - MovieNum
   - 장르
   - 국적
   - 시간
   - 년도

* 책 
   - 제목
   - BookNum
   - 분류

* 추천
   - 회원.ID
   - Num
   - 추천 수
   - 추천 날짜
